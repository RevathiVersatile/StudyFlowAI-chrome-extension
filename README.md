# StudyFlowAI

> Learn Faster. Remember Forever.

StudyFlow AI is an AI-powered Chrome Extension designed to improve learning productivity.

It converts online lessons into:

-  AI Summaries
-  Flashcards
-  Mind Maps
-  Interactive Quizzes
-  AI Chat
-  Revision Notes

## Project Goals

- Reduce study time
- Improve memory retention
- Simplify technical concepts
- Help students prepare for interviews

## Tech Stack

- HTML
- CSS
- JavaScript
- Chrome Extension (Manifest V3)
